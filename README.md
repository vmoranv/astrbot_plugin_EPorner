# AstrBot EPorner 插件

AstrBot 的 EPorner 视频信息查询插件，支持获取视频信息、搜索视频、查看演员信息等功能。

## 功能特性

- 📹 **视频信息查询** - 获取视频标题、观看数、评分、发布日期、时长、标签等信息
- 🔍 **视频搜索** - 按关键词搜索视频
- 👤 **演员信息** - 获取演员详细资料（姓名、订阅者、排名、视频数等）
- 📂 **分类浏览** - 按分类获取视频列表
- 🖼️ **图片模糊处理** - 支持对缩略图进行模糊处理，保护隐私
- 🌐 **代理支持** - 支持配置代理地址访问 EPorner
- 🧹 **自动清理** - 每次发送前自动清理上一次的缓存文件

## 使用方法

### 获取视频信息

```
/ep_video <视频ID或URL>
```

**示例：**
```
/ep_video Y2yAzxsnJ9y
/ep_video https://www.eporner.com/video-Y2yAzxsnJ9y/...
```

### 搜索视频

```
/ep_search <关键词>
```

**示例：**
```
/ep_search latina
```

### 获取演员信息

```
/ep_pornstar <演员ID>
```

**示例：**
```
/ep_pornstar nicole-zurich
```

### 获取分类视频

```
/ep_category <分类名>
```

**示例：**
```
/ep_category bigass
```

## 配置说明

### 代理设置

如果需要通过代理访问 EPorner，可以在配置中设置代理地址：

```
eporner_proxy: http://127.0.0.1:7890
```

或使用 SOCKS5 代理：

```
eporner_proxy: socks5://127.0.0.1:1080
```

### 图片模糊程度

控制缩略图的模糊程度，数值越大模糊程度越高：

- `0` - 不模糊
- `1-3` - 轻微模糊
- `4-7` - 中等模糊（推荐）
- `8-10` - 高度模糊
